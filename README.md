# minnesota

Just a bleeding edge gopher server

## Installation

```
opam pin add --yes https://github.com/drwlrsn/minnesota.git
opam install minnesota
```

## Build
```
dune build
```

## Rune
```
_build/default/src/bin/main.exe
```
